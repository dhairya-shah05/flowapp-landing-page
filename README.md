# flowapp-landing-page

An advanced animated landing page for FlowApp website. 

## Features

- A responsive nav-bar styled by applying glassmorphism effects, blurring the bg to look. 
- A hamburger menu button that is displayed on mobile devices.
- A hero section with animated buttons and hover effects. An image on the right with three glass cards floating on top of it. 
- A features section with 6 cards which glow if hovered on and also display borders.
- A testimonials section with three glass cards and user's profile-pic.
- A CTA section with animated gradient background 
- A multi-column footer with social media hadle icons of the company and a copyright tag at the bottom.

## Technologies Used 
- HTML
- Tailwind CSS

## How To Run

- Open 'index.html' in a browser and try interacting with the various elements and cards on the page.
